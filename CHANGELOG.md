# Changelog

Все значимые изменения в проекте Zumba CRM.

## [2.0.0] - 2026-02-28

### ✨ Добавлено

#### Финансовый модуль
- Модель `Payment` — платежи клиентов
- Модель `Expense` — расходы студии с категориями (аренда, зарплата, маркетинг и др.)
- API endpoints для управления финансами
- Отчётность по доходам и расходам

#### Расписание занятий
- Модель `Schedule` — расписание занятий
- Модель `Enrollment` — запись клиентов на занятия
- Управление группами и листами ожидания
- Отслеживание посещаемости

#### Обратная связь
- Модель `Feedback` — отзывы, оценки, NPS, жалобы и предложения
- Привязка к занятиям и клиентам

#### Заявки с сайта (Leads)
- Модель `Lead` — заявки с формы на сайте
- Интеграция с сайтом zumba-site
- Автоматическое создание клиентов при новой заявке
- Статусы: new, contacted, converted, rejected

#### Аналитика
- Модель `Notification` — уведомления клиентов
- Расширенные метрики дашборда
- Отчёты: RFM-сегментация, отток клиентов, эффективность тренеров

#### Инфраструктура
- Docker и Docker Compose конфигурация
- Nginx конфигурация для продакшена
- Инструкция по деплою на Beget VPS (DEPLOY.md)
- Инструкция по интеграции с сайтом (INTEGRATION.md)

### 🔧 Изменено
- Обновлена структура роутеров
- Улучшена схема базы данных
- Добавлены новые enum типы
