## 📝 Заметки

### Технологии
- **Backend:** FastAPI 0.115+, SQLAlchemy 2.0 (async)
- **Frontend:** HTML + HTMX + Alpine.js + Tailwind CSS
- **Database:** SQLite (dev) → PostgreSQL (prod)
- **Deploy:** Docker + Docker Compose + Nginx

### Структура БД
```
users → clients → subscriptions → visits
              ↓
            payments
              ↓
            enrollments → schedules
              ↓
            feedback
              ↓
            leads
              ↓
            notifications
```

### Доступы по умолчанию
- **Логин:** `admin`
- **Пароль:** `admin123` (изменить в production!)
